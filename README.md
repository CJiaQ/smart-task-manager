# Smart Task Manager

AI智能任务管理平台

## 技术栈
Spring Boot
MyBatis
MySQL
Redis
JWT
Swagger
DeepSeek API

## 项目功能
- 用户登录认证（JWT）
- 任务增删改查
- 分页查询
- Redis缓存优化
- AI任务总结
- Swagger接口文档

## 项目结构
controller
service
mapper
entity
config

## 运行方法
1. 启动 MySQL
2. 启动 Redis
3. 运行 SpringBoot 项目
4. 打开接口文档

http://localhost:8080/doc.html
